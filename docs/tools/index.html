Self-Hosted AI Tools

A curated directory of open-source tools for running AI on your own hardware.

Ollama

Run LLMs locally with a simple CLI and REST API. Supports hundreds of models with GPU acceleration.

ollama.com →

Open WebUI

ChatGPT-like interface for Ollama and any OpenAI-compatible backend. RAG, voice, images, multi-model.

openwebui.com →

vLLM

High-throughput LLM serving with PagedAttention. Optimized for production inference at scale.

GitHub →

LiteLLM

Unified API proxy for 100+ LLM providers. Load balancing, rate limiting, cost tracking, and fallbacks.

GitHub →

ComfyUI

Node-based image and video generation UI for Stable Diffusion and FLUX models.

GitHub →

llama.cpp

C++ inference engine for LLMs. Powers Ollama's GGUF backend. Runs on CPU, GPU, and Apple Silicon.

GitHub →

Continue

Open-source AI code assistant for VS Code and JetBrains. Supports Ollama, OpenAI, and custom backends.

GitHub →

RAGFlow

Deep document understanding RAG engine. Parses complex layouts, tables, and images for accurate retrieval.
